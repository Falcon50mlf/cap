# Cap&rsquo;

> Donne-toi un cap.

Plateforme de découverte des écoles de commerce et de leurs débouchés, à travers des mini-jeux qui simulent le vrai monde pro et les vrais cours.

## Quickstart

```bash
cp .env.example .env.local   # remplis tes clés Supabase
npm install
npm run dev                  # http://localhost:3000
```

Stack : Next.js 14 (App Router) · TypeScript · Tailwind · Framer Motion · Zustand · Supabase.

Voir [`CLAUDE.md`](./CLAUDE.md) pour la spec produit complète et [`STATUS.md`](./STATUS.md) pour l&rsquo;avancement.
